# 起源
关于我为什么启动这个项目，因某晚觉得c++新建一个项目真的好麻烦，也没有好用的脚手架，linux和windows项目差别也是巨大，还得去找各种第三方工具库等等麻烦事儿
所以我准备在这个项目里使用cmake作为跨平台，收录我自己认为比较好的代码风格，编译模板，第三方库，以及自己常用的工具类
